# IX-Ray Hub

[English](./README.md) | Русский

## Требования

- Visual Studio 2022, Visual Studio Code или Rider
  - Avalonia плагин
- .NET 8 SDK
- Git

## Сборка

- Скачать репозиторий:

  ```console
  git clone https://github.com/ixray-team/ixray-modhub
  ```

- Собрать `IXRay.Hub.sln` используя IDE или команду:

  ```console
  dotnet build IXRay.Hub.sln
  ```
